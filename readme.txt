# PDF Okuyucu

PDF Okuyucu, kullanıcıların PDF dosyalarını seçmelerine ve bu dosyaların içeriğini sesli olarak okumalarına olanak tanıyan basit bir Python uygulamasıdır.

## Gereksinimler

- Python 3.x
- `tkinter` modülü
- `pyttsx3` modülü
- `PyPDF2` modülü

Gerekli modülleri yüklemek için terminal veya komut istemcisinde aşağıdaki komutu kullanabilirsiniz:

pip install pyttsx3 PyPDF2


## Kullanım

1. Kodu çalıştırın: `python pdf_okuyucu.py`
2. "PDF Dosyası Seç" düğmesine tıklayarak bir PDF dosyası seçin.
3. Seçilen PDF dosyasının içeriği sesli olarak okunacaktır.
